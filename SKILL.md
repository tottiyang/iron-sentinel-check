## 简介

**铁血哨兵** 是 A股买点智能审核系统，基于 11 项严格标准判断股票买点时机，综合评估后给出买入/观望/不推荐建议。

---

## 核心审核维度（11项）

| # | 审核项 | 权重 | 通过条件 |
|---|--------|------|---------|
| 1 | MACD动能增强 | 10% | 红柱增长 或 绿柱缩短 |
| 2 | 量能放大 | 10% | 今日成交量 > 近5日均量×1.1 |
| 3 | 主力净流入 | 10% | 超大单+大单净流入 > 0 |
| 4 | 上升趋势 | 10% | MA5 > MA10 > MA20 且价格在MA5上方 |
| 5 | 基本面良好 | 10% | PE 0~150 且 PB < 10 |
| 6 | 分时均线上方 | 10% | 现价 ≥ 均价 |
| 7 | 大盘日内非下行 | 5% | 上证或深证涨跌幅 > -0.5% |
| 8 | 大盘趋势非下行 | 5% | 上证指数 > MA5 |
| 9 | 板块趋势向上 | 5% | 所属板块涨幅 > 0 |
| 10 | 龙头活跃 | 12% | 近5日正增≥3/5 且 今日盘中正增≥3/5 |
| 11 | 筹码集中度 | 13% | 股东户数环比<0 且 前十流通股≥30% |

## 评分等级

| 等级 | 分数 | 含义 | 操作建议 |
|------|------|------|---------|
| ⭐ 优质买点 | ≥90 | 各项指标健康 | 强烈建议关注买入 |
| ✅ 良好买点 | 80-89 | 大部分指标达标 | 可以考虑买入 |
| 🟡 一般买点 | 70-79 | 条件参差 | 可少量试探 |
| ⚠️ 较差买点 | 60-69 | 风险项多 | 建议观望等待更好时机 |
| ❌ 不推荐买入 | <60 | 多项不符 | 耐心等待更好时机 |

## 使用方式

### 方式一：直接导入（推荐）

```python
import sys
sys.path.insert(0, '~/.qclaw/skills/iron-sentinel/scripts/analysis')

from engine import IronSentinelEngine
from build_report import build_from_report

report = IronSentinelEngine('300438').audit()
print(build_from_report(report))
```

### 方式二：命令行

```bash
cd ~/.qclaw/skills/iron-sentinel/scripts/analysis

python3 engine.py 300438
```

### 方式三：快速脚本

```bash

```

## 数据源策略（三层降级）

```
优先: NeoData（质量最高）
  ↓ 失败 → 备用: 腾讯/新浪（实时行情）
      ↓ 失败 → 降级: AkShare（免费兜底）
          ↓ 失败 → 标记"数据不可用"（不影响其他项）
```

- 每个数据源失败时明确标记，不静默跳过
- 数据可用率正常应达到 7-9/9 项

## 重要提示

1. **本系统仅供参考**，不构成投资建议
2. **请到海通证券确认 KD 指标**后再做最终决策
3. **大盘环境不好时**，即使个股评分高也应谨慎
4. **数据不可用项越多**，评分可靠性越低

## 数据完整性保护

- **SHA256 指纹**: engine.py 输出时对 `results` + `leader_details` 生成指纹，防止报告篡改
- **防篡改校验**: analyze.py 和 build_report.py 入口校验指纹，被篡改则拒绝处理
- **原始数据锁定**: 展示层表格数据直接来自 `raw_data`，不得修改

---

## 非交易日处理

- 周末/节假日自动使用**最后交易日数据**
- 强制走"盘后模式"，量能审核使用历史数据参与计算
